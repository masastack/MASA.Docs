# 后台任务 - Hangfire

## 概述

基于 [Hangfire]([Hangfire – Background jobs and workers for .NET and .NET Core](https://www.hangfire.io/)) 实现的后台任务，支持一次性任务和周期性任务

1. 安装 `Masa.Contrib.Extensions.BackgroundJobs.Hangfire`
