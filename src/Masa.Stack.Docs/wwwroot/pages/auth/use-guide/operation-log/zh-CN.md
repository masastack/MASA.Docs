# 操作日志

操作日志用于记录Auth后台的操作记录，包含Auth后台各页面的操作记录、用户登录。

### 列表

操作日志列表有分页、模糊搜索、时间范围搜索、相对时间搜索、操作类型筛选、操作人筛选功能。

> 模糊搜索支持操作人和操作描述
